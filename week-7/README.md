# Week 7 - React Problems (100xDevs)

This folder contains the React assignments and practice problems for Week 7 of the 100xDevs course.

## Assignments

### 1. Profile Card
Create a reusable profile card component in React.

**Concepts Covered:**
- Functional Components
- Props
- CSS Styling
- Component Reusability

---

### 2. Background Changer
Create a React application that changes the background color dynamically.

**Concepts Covered:**
- useState
- Event Handling
- Dynamic Styling

---

### 3. Custom React using JavaScript
Build a simple custom React renderer using plain JavaScript.

#### Requirements

1. Create an object representing an HTML anchor element.
2. Create a function that converts the object into an HTML element.
3. Create a `customRender()` function that renders the element inside a root container.

**Concepts Covered:**
- DOM Manipulation
- JavaScript Objects
- Custom Rendering

---

### 4. Paragraph Generator
Create a paragraph generator that accepts the number of words and generates a paragraph.

**Concepts Covered:**
- useState
- Forms
- Dynamic Rendering

---

### 5. GitHub Profile Card
Fetch GitHub user information using the API below and display it as a profile card.

```
https://api.github.com/users/${username}
```

**Concepts Covered:**
- Fetch API
- Async/Await
- useEffect
- Conditional Rendering

---

### 6. OTP Login
Create an OTP Login interface similar to Hotstar.

#### Requirements

- OTP input fields
- Automatically move the cursor to the next input after entering a digit
- Handle complete OTP input

**Concepts Covered:**
- useRef
- Controlled Inputs
- Focus Management
- Event Handling

---

### 7. Birthday Wisher
Create a birthday wishing website.

#### Requirements

- Accept a person's name.
- Display 2–3 birthday wish cards dynamically.

**Concepts Covered:**
- Props
- Components
- Conditional Rendering
- Dynamic UI
